# 大致推荐框架

## 召回部分:
实行多路召回 一共四路:
1.用户兴趣标签
2.协同过滤
3.物品新颖度
4.物品热度/流行度


## 排序部分：

将召回部分得到的总召回，以其对应的4中特征值标准化处理后进入DeepFM中进行CTR预测，根据预测值得到最终排序
再通过用户对推荐内容的反馈对其生成标签产生训练数据从而针对不同用户个性化地训练了DeepFM。
